# AetherFlow - AI 手势驱动粒子交互特效系统

[![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Web-orange.svg)]()
[![FrameRate](https://img.shields.io/badge/FPS-60-brightgreen.svg)]()

**AetherFlow** 是一款基于浏览器、高度拟真的实时 AI 手势交互粒子物理特效系统。它通过摄像头捕获用户双手，使用 Google MediaPipe Hands 算法实时定位 21 个手部骨骼关键点，并在 HTML5 Canvas 上渲染高流畅度、强交互感的物理粒子流光。系统内置多种拟真特效主题，并支持 60 FPS 本地高清录屏与多格式自适应导出。

---

## ✨ 核心特性

- **🎬 60 FPS 高清本地录像**：通过 Canvas 双图层融合技术，将摄像头视频帧与粒子、手势骨骼在 Canvas 底层同步镜像绘制。支持使用 `MediaRecorder` 导出包含真人背景与特效的 60 帧极速丝滑视频。
- **📁 多格式自适应导出**：提供 **WebM、MP4、AVI** 格式选择。支持自动嗅探浏览器硬件 H.264/MP4 编码，若不支持则以高画质 WebM 录制并自适应重映射扩展名，确保主流本地播放器（VLC、QuickTime 等）双击即可播放。
- **💥 跨手协同联动（双手互动）**：
  - **指尖高压电弧 (Lightning Arc)** ⚡：双手食指伸出且拉开距离时，在指尖拉出实时弯折、带随机电火花溅射的跳动蓝色电弧。
  - **超新星碰撞大爆炸 (Supernova)** 💥：双手掌心靠近（距离 < 110px）的瞬间在中心点引爆超新星，释放同心激波冲击环并将场上所有粒子暴力吹飞，腾起 320+ 颗漫天飞散的金粉。
- **🎨 四大高拟真物理主题**：
  - 🔥 **烈焰风暴 (Fire Storm)**：炽红火星伴随受热力上升的气流，生命后期会真实膨胀并转化为逐渐消散的半透明灰色烟尘。
  - 🌌 **极光魔幻 (Aurora Magic)**：如绸带般优雅舞动的蓝绿紫气体粒子，伴随做周期性明暗闪烁的亮白恒星尘。
  - ⚡ **霓虹火花 (Neon Spark)**：带运动模糊线段（Motion Blur Streaks）的抛光金属电浆火花，受重力下坠并在屏幕底部进行弹性物理碰撞反弹。
  - 🌀 **虚空引力 (Void Gravity)**：模拟黑洞吸积盘，粒子自转速度遵循真实的开普勒轨道切变（越靠近拳心自转越快），且在进入事件视界（掌心 8px）后会被无情吞噬湮灭。
- **📊 实时运行诊断面板**：玻璃魔态（Glassmorphism）面板实时监控 FPS 帧率、粒子存活总数、当前活动手部数量和手势识别状态（左手/右手分类诊断）。
- **⚙️ 深度物理调参**：支持分别调节每个主题的粒子尺寸，并在引力场滑块中支持 **0.0x（无引力）到 ∞（无穷大）** 的非线性映射调节，拉满即可生成瞬间吸附一切的“引力奇点”。

## 🚀 快速开始与使用步骤

项目采用纯原生前端 ESM 架构，**无需任何构建步骤，开箱即用**。

### 1. 克隆/下载项目
将本仓库代码克隆或下载到本地：
```bash
git clone https://github.com/Eser-Tired/Partical_effects.git
cd Partical_effects
```

### 2. 启动本地静态服务器
由于浏览器安全策略，摄像头调用与 MediaPipe WebAssembly 模型加载必须运行在本地服务器环境（`localhost`）或安全链接（`HTTPS`）下。

您可以使用以下任一极其简便的方法启动本地服务：

* **方法 A：使用 Python（推荐，系统自带）**
  ```bash
  python3 -m http.server 8000
  ```
* **方法 B：使用 Node.js (`http-server`)**
  ```bash
  npx http-server -p 8000
  ```
* **方法 C：使用 VS Code Live Server 插件**
  - 在 VS Code 中打开该项目文件夹，点击右下角的 **Go Live**。

### 3. 打开网页进行体验
- 启动服务后，使用现代浏览器（推荐 **Google Chrome** 或 **Microsoft Edge**）访问：
  **[http://localhost:8000](http://localhost:8000)**
- 授权浏览器使用**摄像头权限**。
- 等待 1-2 秒（AI 模型首次从 jsDelivr CDN 加载初始化），看到加载屏幕消失、“系统就绪”且右上角 FPS 开始跳动，即可摆出各种手势体验！

## 🛠️ 技术亮点与实现逻辑

1. **JIT 粒子发光精灵缓存（Sprite Cache）**：
   Canvas 绘制复杂的软边缘径向渐变（Radial Gradient）非常消耗 CPU。我们在系统初始化和颜色突变时，即时（JIT）在离屏 Canvas 上将各个色彩通道编译为高清模糊气团 Sprite。运行时利用硬件加速的 `drawImage` 绘制精灵图，配合 `lighter`（Screen 混色）混合模式，以最低成本渲染体积感十足的发光流体。
2. **线段拉伸模拟运动模糊（Motion Blur Streaks）**：
   对于金属火花，粒子通过连接当前位置 `(x, y)` 与 `(x - vx * 1.5, y - vy * 1.5)` 绘制。线段长度与速度大小成正比，利用 Canvas 硬件加速的高速 stroke 线段绘制，在 2D Canvas 上呈现出电影级的相机快门运动模糊效果。
3. **迟滞区间判定（Hysteresis Threshold）**：
   为了防止双手在靠近大爆炸临界点时，由于识别微小抖动造成大爆炸每秒重复触发百次（导致卡顿和死循环），我们引入了迟滞防抖逻辑：双手距离低于 `110px` 触发引爆并锁定；只有当双手重新拉开距离超过 `150px` 后，引爆锁定才会解除。
4. **旋转无关的手指伸缩检测**：
   传统 MediaPipe 手指检测依靠 Y 轴坐标对比（如指尖 Y 是否小于 MCP 关节 Y），这在手倒置或横着时会彻底失效。我们重构为**腕部距离比例算法**：计算指尖到手腕的距离是否显著大于第二指节（PIP）到手腕的距离。由于手指弯曲时指尖必定拉近与腕部的距离，该算法实现了完美的**旋转无关性**。
